
# ⚖️AI LEGAL ASSISTANT SYSTEM

A web-based AI Legal Assistant System designed to provide legal support for Public Users, Lawyers, and Police Officers.
This system helps users understand legal procedures, manage case records, handle complaints, and access legal guidance through an intelligent and user-friendly platform.

# 📌 Project Overview

The AI Legal Assistant System is developed to simplify access to legal information and improve the management of legal activities.
It provides role-based dashboards and features for different users.

Public Users:

* Legal guidance
* Rights awareness
* Procedure help
* Evidence assistance
* IPC section exploration

Lawyers:

* Case upload and management
* Case history tracking
* Case analytics
* Legal report generation
* Lawyer AI chatbot

Police Officers:

* Complaint entry
* Complaint details management
* Investigation notes
* Evidence checklist
* Case status tracking
* Police AI chatbot

The system is built using Python Flask, with HTML, CSS, and JavaScript for the frontend, and SQLite for data storage.

## Installation
  
1. Clone the repository

git clone [https://github.com/your-username/ai-legal-assistant-system.git](https://github.com/your-username/ai-legal-assistant-system.git)

2. Go to the project folder

cd ai-legal-assistant-system

3. Create a virtual environment

python -m venv venv

4. Activate the virtual environment

For Windows:
venv\Scripts\activate

For Mac/Linux:
source venv/bin/activate

5. Install required dependencies

pip install -r requirements.txt

6. Run the Flask application

python app.py

7. Open in browser

[http://127.0.0.1:5000/](http://127.0.0.1:5000/)

##  ⚠️ Limitations
*The system acts as a support tool and not as a replacement for professional legal judgment
* Final legal decisions must be made by qualified legal authorities
* SQLite may not be suitable for very large-scale deployment
* Accuracy depends on stored legal datasets and knowledge resources
* Some advanced real-time legal features are not included yet
